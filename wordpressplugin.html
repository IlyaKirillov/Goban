Plugin for WordPress

Installation

  1. Download the archive
  2. Unzip the archive into your /wp-content/plugins/ directory
  3. Activate the plugin through the 'Plugins' menu in WordPress

Usage

The plugin add new shortcode to the WordPress instllation:[webgoboard]...[/webgoboard]

Loading file

There are two ways to load file: by url from your site (it is not possible to load a file from a remote server due to security reasons) or to type in the SGF-data directly inside the shortcode block.
